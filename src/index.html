OpenLog

2015 - CURRENT

An easy way to aggregate client side logging
OpenLog helps you track down client side javascript errors that you can never seem to replicate on your machine. Errors and stack traces that happen on any machine will be readily available to you via a web interface. This project was made because their were some errors users were reporting that I simply couldn't replicate, with this system setup I could see their errors as they happened with a full trace log. This vastly improved my error debug and fix time.
Melbourne, Australia
